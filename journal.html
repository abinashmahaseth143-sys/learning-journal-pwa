Week 2 – Frontend Fundamentals

1. How did you approach mobile-first design?
I started by designing for small screens first, using simple layouts that scale up with media queries.

2. What was the most useful HTML or CSS concept you applied this week?
Flexbox was very useful for aligning elements and building a responsive navigation bar.

3. What part of HTML or CSS did you find most challenging?
I found responsive breakpoints a bit tricky to adjust for different screen sizes.
